# DuckMCLauncher (DML)

🦆 **DuckMCLauncher** - A modern Minecraft Java Edition launcher for Android based on FCL [Fold Craft Launcher](https://github.com/FCL-Team/FoldCraftLauncher).

## Features

✅ **Full Minecraft Support**
- All Minecraft Java versions (including snapshots)
- Mod loaders: Forge, NeoForge, LiteLoader, OptiFine, Fabric, Quilt, Cleanroom
- Multiple Java versions (8, 17, 21, 25)

✅ **Advanced Features**
- Virtual mouse and custom key mapping
- Shader support (VirGL/Zink/MG renderers)
- Dynamic resource management (mods, modpacks, textures, shaders, worlds)
- Theme customization
- Renderer plugin system

✅ **User-Friendly**
- Clean, minimalist UI design
- Easy mod management
- Quick game launch
- Profile management

## Based On

- **FCL (Fold Craft Launcher)** - Original launcher engine
- **HMCL** - Launcher core functionality
- **PojavLauncher** - Java runtime for Android
