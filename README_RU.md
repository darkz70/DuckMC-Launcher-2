# DuckMCLauncher (DML)

🦆 **DuckMCLauncher** - современный лаунчер Minecraft Java Edition для Android на базе движка FCL (Fold Craft Launcher).

## Возможности

**Полная поддержка Minecraft**
- Все версии Minecraft Java (включая снимки)
- Загрузчики модов: Forge, NeoForge, LiteLoader, OptiFine, Fabric, Quilt, Cleanroom
- Несколько версий Java (8, 17, 21, 25)

**Продвинутые функции**
- Виртуальная мышь и пользовательское сопоставление клавиш
- Поддержка шейдеров (VirGL/Zink/MG рендеры)
- Динамическое управление ресурсами (моды, модпаки, текстуры, шейдеры, миры)
- Настройка темы
- Система плагинов рендеров

**Удобство использования**
- Чистый минималистичный дизайн UI
- Легкое управление модами
- Быстрый запуск игры
- Управление профилями

## На основе

- **FCL (Fold Craft Launcher)** - оригинальный движок лаунчера

- **PojavLauncher** - Java runtime для Android
