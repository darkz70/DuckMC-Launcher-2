# DuckMCLauncher (DML)

🦆 **DuckMCLauncher** - 基于FCL (Fold Craft Launcher)引擎的现代Android Minecraft Java版启动器。

## 功能特性

**完整的Minecraft支持**
- 所有Minecraft Java版本（包括快照版）
- Mod加载器：Forge、NeoForge、LiteLoader、OptiFine、Fabric、Quilt、Cleanroom
- 多个Java版本（8、17、21、25）

**高级功能**
- 虚拟鼠标和自定义按键映射
- 着色器支持（VirGL/Zink/MG渲染器）
- 动态资源管理（Mod、模组包、纹理、着色器、世界）
- 主题自定义
- 渲染器插件系统

**用户友好**
- 简洁的极简UI设计
- 轻松的Mod管理
- 快速游戏启动
- 档案管理

## 基于

- **FCL (Fold Craft Launcher)** - 原始启动器引擎
- **HMCL** - 启动器核心功能
- **PojavLauncher** - Android上的Java运行时
